# US-10 — Show or hide my lines during a run

**Epic:** Practice Session

## User Story
As an actor, I want to toggle whether my lines are shown on the cue card so that I can use the app as a prompt early in learning and test myself later.

## Acceptance Criteria
- Toggle accessible before starting a run
- "Training wheels on": line text shown on cue card
- "Training wheels off": only the preceding line shown, my line hidden
