# US-01 — Import a plain text script

**Epic:** Script Management

## User Story
As an actor, I want to import a `.txt` file from the Files app so that I can load my script without typing it manually.

## Acceptance Criteria
- Accepts share sheet and document picker
- Parses `CHARACTER NAME: line text` and `CHARACTER NAME\nLine text` formats
- Shows a preview of parsed lines before confirming
